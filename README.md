# Puppy

## Setup

```bash
# pnpm
pnpm install --registry=https://registry.npmmirror.com/

pnpm up --latest
```

## Development Server

```bash
# pnpm
pnpm dev
```

## Production

```bash
# pnpm
pnpm build
```

Locally preview production build:

```bash
# pnpm
pnpm preview
```

## Iconify

```shell
pnpm i @iconify-json/{collection_name}
```
